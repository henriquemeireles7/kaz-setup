## Core Principles

1. **Maturity levels drive evolution** — Every category progresses L1→L5. No skipping.
2. **Measure before improving** — Know your current level before targeting the next.
3. **Bottleneck-first** — Fix the lowest-scoring category that blocks the most others.
4. **Error feedback loop** — Every error produces a prevention artifact. Zero recurring errors.
5. **Agent-as-execution** — AI agents execute, humans decide strategy and taste.

## 5 Universal Maturity Levels

| Level | Name | Definition |
|-------|------|------------|
| L1 | Manual | Human does everything. No automation. |
| L2 | Assisted | AI helps but human drives. Templates, suggestions. |
| L3 | Automated | AI executes with human approval gates. Pipelines. |
| L4 | Self-Learning | System improves from outcomes. Feedback loops close automatically. |
| L5 | Self-Evolving | System creates new capabilities. Writes its own rules. |

Each category scores 0-99:
- L1 (0-19): Manual
- L2 (20-39): Assisted
- L3 (40-59): Automated
- L4 (60-79): Self-Learning
- L5 (80-99): Self-Evolving

Company score = sum of all category scores.
